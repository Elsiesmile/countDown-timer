# Проект: Таймер обратного отсчёта.

### [Ссылка на проект](https://elsiesmile.github.io/countdown-timer/)

<p>Приложение для отсчета времени до нужной даты, с возможностью ввода названия и выбора даты. При обновлении страницы таймер продолжает работу.</p>

### Приложение запускается командой:

```
npm run serve
```
